Texture pack manager for MooMoo.io.
This script allows you to change textures easier and comfortable, press O to open menu.

## How to use:
- Copy and paste Default-Version or Inject-Version of Texture-Pack manager in tampermonkey.
- Join in game.

## Notice:
- If you are using Inject-Version, you should download Allow-CORS extension.
- Link: https://chromewebstore.google.com/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf
